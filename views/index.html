ART MART API ENDPOINTS

User Endpoints

GET /users Shows List of all users
GET /users/:id Shows user with the specific id
POST /users Registers user on the database
PUT /users/:id Updates users with the specific id on the database
DELETE /users/:id Deletes user with the specific id from the database

Product Endpoints

GET /products Shows all available products
GET /products/:id Gets the product with the specific id
POST /products Adds a product to the database
PUT /products/:id Edits product with the specific id on the database
DELETE /products/:id Deletes product with the specific id

Painting Endpoints

GET /paintings Shows all available paintings
GET /paintings/:id Gets the paintings with the specific id
POST /paintings Adds a paintings to the database
PUT /paintings/:id Edits paintings with the specific id on the database
DELETE /paintings/:id Deletes paintings with the specific id

User Cart Endpoints

GET /users/:id/cart Shows all items in the cart of the user with the specific id
POST /users/:id/cart Updates the items in the cart of the user with the specific id
DELETE /users/:id/cart Clears the cart of the user with the specific id
